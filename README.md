# Dragons of Mugloar

A Vue 3 web app built on top of the [Dragons of Mugloar](https://dragonsofmugloar.com) REST API. Take on quests, earn gold, and survive long enough to reach 1000 points.

See [ARCHITECTURE.md](ARCHITECTURE.md) for design decisions and tech stack.

## Setup

```bash
npm install
```

## Running

```bash
npm run dev
```

## Testing

```bash
npm test
```
